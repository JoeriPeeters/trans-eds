# Website Specifications

This directory contains detailed specifications for all features of the TransGlobal logistics website.

## Spec-Based Development Approach

Each spec follows this structure:
1. **Overview** - What the feature does
2. **Figma Reference** - Links to specific design elements
3. **Requirements** - Functional and technical requirements
4. **Adobe EDS Blocks** - Specific blocks/components needed
5. **Content Structure** - Document/spreadsheet structure
6. **Responsive Behavior** - Mobile, tablet, desktop
7. **Acceptance Criteria** - Definition of done

## Testing Checklist

For each spec, verify:
- [ ] Visual match with Figma design
- [ ] Responsive behavior on all breakpoints
- [ ] Content editable via Adobe EDS documents
- [ ] Accessibility (WCAG AA minimum)
- [ ] Performance (Lighthouse score > 90)
- [ ] SEO metadata present
- [ ] Cross-browser compatibility
