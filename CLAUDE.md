# Transport Company Website POC - Project Context

## Project Overview

Building a POC website for an independent global transportation and logistics company competing with industry leaders like CEVA Logistics and MAERSK.

**Company Name:** TransGlobal (placeholder)

## Primary Goals

1. **Improve Online Visibility**
   - SEO optimization for global discoverability
   - AI discoverability support
   - Ensure searchable and findable across all target markets

2. **Support Global Growth**
   - ~90 countries presence
   - Local contacts easily reachable in each market
   - Focus on Top 15 customer countries

3. **Dual Audience Strategy**
   - **Existing clients:** Already knowledgeable, need quick access to services/contacts
   - **Smaller/new clients:** Require educational content to understand offerings

## Important Notes

### Design Decisions Made
1. **Dark navy backgrounds** for header, footer, hero, "Why Choose Us" to create visual rhythm and emphasis
2. **White and light gray backgrounds** for content-heavy sections (Services, Industries)
3. **Blue gradient for CTA section** to stand out and drive action
4. **Two-column hero layout** balancing messaging with visual interest
5. **Icon + text pattern** used consistently across service and benefit cards
6. **Trust indicators early** (client logos, statistics) to build credibility immediately

### Accessibility Considerations
- White text on navy: Contrast ratio 14.7:1 (WCAG AAA)
- All buttons have minimum 44x44px touch target
- Service/benefit cards have clear visual hierarchy
- Icons supplement text, not replace it

### Performance Considerations
- SVG for all vector graphics (logo, icons)
- Optimize container ship illustration (can be exported as optimized SVG)
- World map uses basic shapes (performance-friendly)
- Emoji fallback for industry icons (can be replaced with SVG if needed)
